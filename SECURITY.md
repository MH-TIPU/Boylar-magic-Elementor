## Security Policy

### Supported Versions
- **0.1.x**: supported

### Reporting a Vulnerability
If you discover a security issue, please contact the plugin owner/maintainer privately.

Include:
- Plugin version
- Steps to reproduce
- Impact assessment (who can exploit, what they gain)

Please do not post publicly until a fix is released.

### Data shared with third parties
When a user clicks **Generate Section**, the plugin sends the prompt (and optionally a reference image) to the configured AI provider endpoint.
